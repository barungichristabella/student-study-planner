About This Project

Learn more about the Student Study Planner and why it was built.

What is the Student Study Planner?

The Student Study Planner is a free, browser-based web application designed to help students manage their assignments, topics and deadlines in one simple place. It was built as part of an Internet Technologies and Web Design course at Cavendish University Uganda.

The Problem

Many students struggle to keep track of their assignments and deadlines. They either forget tasks, study at the last minute, or have no simple tool designed specifically for students. This planner was created to solve that problem.

Features

📝 Task Management

Add tasks with subject, topic, start date, deadline, priority and status all in one form.

🔔 Deadline Reminders

Automatic popup and banner reminders warn you when a task is due today, tomorrow or overdue.

💾 Auto Save

Tasks are saved automatically in your browser so they stay even after you close the page.

📱 Mobile Friendly

The website works on both desktop and mobile screens thanks to responsive design.

Technologies Used

This project was built using HTML5, CSS3 and JavaScript — no frameworks or libraries were used.
